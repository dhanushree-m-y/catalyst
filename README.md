# Catalyst

A community website for **Catalyst** — a movement where women across every field
(software, electronics, civil, mechanical, design and beyond) belong, learn and grow.

Built with **Next.js (App Router) + TypeScript**.

## Pages

- **`/`** — Landing page: hero, community stats, about, why, founder, Git With Her
  teaser, upcoming events, "You belong here" carousel, sponsors, host an event, contact.
- **`/git-with-her`** — Dedicated hackathon page: cinematic space hero, quick-facts
  bar, problem statements, eligibility, prizes, build-day roadmap, speakers, FAQ, register.

## Highlights

- Pink editorial theme with wave-divider bands and script display headings
- Scroll-reveal animations, animated stat counters, scrollspy nav
- Magic-cursor sparkle trail, scroll progress bar + back-to-top
- Auto-playing image carousel
- SEO event structured data (JSON-LD) on the hackathon page

## Getting started

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # production build
```
